# StreetEats 😋

Find the best street food near you — curated from top food bloggers and YouTube creators.

## What it does

StreetEats surfaces famous street-food spots featured in food vlogs, so you can discover where locals (and creators) actually eat.

## Tech stack

- **Next.js** — app framework
- **Supabase** — spots data
- **Tailwind CSS** — styling

## Getting started

1. Clone the repo and install dependencies:

   ```bash
   npm install

2. Add your Supabase keys to .env.local:

NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key

3. Run the dev server:
npm run dev

4. Open http://localhost:3000
